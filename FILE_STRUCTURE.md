## 1. 完整目录树

```
crabcoder/                                    # 项目根目录
├── CLAUDE.md                                 # Claude Code 项目指南
├── CRABCODER_ARCHITECTURE.md                 # 架构设计文档
├── DEVELOPMENT_PLAN.md                        # 开发规划文档
├── FILE_STRUCTURE.md                         # 本文档
├── go.mod                                    # Go 模块定义
├── go.sum                                    # Go 依赖锁定
├── Makefile                                  # 构建脚本
│
├── cmd/                                      # 命令行入口
│   ├── cli/                                  # CLI 主程序
│   │   └── main.go                           # 程序入口
│   └── repl/                                 # REPL 交互循环
│       └── repl.go                           # 交互式命令行
│
├── pkg/                                      # 核心代码包
│   ├── core/                                 # 核心模块
│   │   ├── app/                             # 应用构建器
│   │   │   ├── app.go                       # App 主结构
│   │   │   ├── config.go                     # 配置管理
│   │   │   ├── errors.go                     # 错误定义
│   │   │   └── middleware.go                 # 中间件
│   │   ├── bus/                             # 消息总线
│   │   │   └── bus.go                        # 事件发布/订阅
│   │   └── state/                           # 状态管理
│   │       └── state.go                      # 状态存储
│   │
│   ├── service/                              # 服务层
│   │   ├── ai/                              # AI 客户端
│   │   │   └── client.go                     # 模型客户端
│   │   ├── tool/                            # 工具协调
│   │   │   └── coordinator.go                 # 工具执行协调
│   │   └── permission/                      # 权限管理
│   │       └── permission.go                 # 权限检查
│   │
│   ├── terminal/                             # 终端模块
│   │   └── terminal.go                      # 终端接口
│   │
│   └── tools/                               # 工具系统
│       ├── tool.go                          # 工具接口和注册表
│       ├── base/                            # 基础工具
│       │   └── read.go                      # 读取文件工具
│       └── exec/                            # 执行工具

├── configs/                                  # 配置文件示例
│   └── config.yaml                          # 主配置文件
│
└── openspec/                                 # OpenSpec 规范
    ├── config.yaml                           # OpenSpec 全局配置
    └── schemas/
        └── tdd-driven-v2/                   # TDD Schema
            ├── schema.yaml                  # Schema 定义
            └── templates/                   # 模板目录
                ├── proposal.md              # 变更提案模板
                ├── spec.md                 # 行为规格模板
                ├── design.md              # 技术设计模板
                ├── tasks.md               # TDD 任务模板
                └── plan.md               # 执行计划模板
```

## 5. OpenSpec 工作流文件生成链

```
用户输入
    │
    │ /opsx:propose "功能描述"
    │
    ▼
┌────────────────────────────────────────────────────────────┐
│  读取 openspec/schemas/tdd-driven-v2/schema.yaml         │
│  读取 openspec/config.yaml (context)                      │
└────────────────────────────────────────────────────────────┘
                          │
    ┌─────────────────────┼─────────────────────┐
    ▼                     ▼                     ▼
┌─────────┐         ┌─────────┐           ┌─────────┐
│proposal │         │  spec   │           │ design  │
│ .md    │         │  .md    │           │  .md   │
└─────────┘         └─────────┘           └─────────┘
    │                   │                     │
    └───────────────────┴─────────────────────┘
                        │
                        ▼
┌────────────────────────────────────────────────────────────┐
│  生成到 openspec/changes/<change-name>/                    │
│                                                             │
│  <change>/                                                  │
│  ├── proposal.md                                            │
│  ├── spec.md                                                │
│  ├── design.md                                              │
│  ├── tasks.md                                               │
│  └── plan.md                                                │
└────────────────────────────────────────────────────────────┘
                        │
                        │ /opsx:apply <change-name>
                        ▼
┌────────────────────────────────────────────────────────────┐
│  执行 plan.md 中的每个 TDD task                            │
│                                                             │
│  RED  → GREEN  → REFACTOR (可选)                           │
│   │        │                                                │
│   └────────┴─────► 单元测试通过                             │
│                                                             │
│  产出: pkg/ 下的实现代码 + 测试代码                         │
└────────────────────────────────────────────────────────────┘
```

## 7. 标签定义

| 标签 | 说明 |
|------|------|
| `<-` | 依赖/引用关系 |
| `──▶` | 导入关系 |
| `│` | 目录层级 |
| `├──` | 同级目录/文件 |
| `└──` | 最后一个同级目录/文件 |
